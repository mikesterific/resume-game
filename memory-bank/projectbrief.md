# Resume Game - Project Brief

## Project Overview
**"Portfolio Quest"** - A retro RPG-inspired interactive interface for a professional website portfolio that integrates résumé and showcases work. The game creates an engaging, animated experience where visitors "explore" your professional world as a character on a quest to discover skills, projects, and résumé.

## Project Goals
1. Create an interactive, game-like interface that balances fun with professionalism
2. Build a 2D pixel-art RPG world with explorable areas showcasing portfolio content
3. Integrate résumé seamlessly into the game experience
4. Optimize for HDMI display on large screens (TV/monitor presentations)
5. Maintain accessibility while providing engaging animations and interactivity

## Key Requirements
### Core Features
- **Interactive Game World**: 2D pixel-art world with areas like "Skill Village," "Project Forest," and "Résumé Tower"
- **Character Control**: User-controlled avatar with keyboard/mouse or touch controls
- **Portfolio Showcase**: Projects as "treasures" or "quests" with modal details
- **Résumé Integration**: Interactive résumé as a "scroll" or "book" in Résumé Tower
- **Animations**: Smooth pixel-art animations, sparkles, fade-ins, walking cycles
- **HDMI Optimization**: High-resolution display support (1080p/4K) for presentations

### Technical Requirements
- Web-based interface using Phaser.js + Vue 3 + Vite
- Responsive design for desktop, tablet, phone, and large displays
- 60 FPS performance for smooth large screen display
- Professional balance with toggle options for sound/music
- Modern Vue 3 Composition API for component architecture

## Success Criteria
- ✅ Engaging, interactive experience that showcases portfolio effectively
- ✅ Professional presentation suitable for business contexts
- ✅ Smooth performance on HDMI-connected large displays
- ✅ Accessible navigation with multiple input methods
- ✅ Fast loading times and optimized performance
