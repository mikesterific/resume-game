# Portfolio Images

This directory contains the artwork images displayed in the Space Museum portfolio frames.

## Image Specifications

### **Aspect Ratio: 1.92:1 (Custom Widescreen)**
All images should maintain a 1.92:1 aspect ratio (width:height ratio of 854:445) to properly fit the museum frames without stretching or cropping.

### **Recommended Resolutions** (in order of preference):

1. **854 × 445 px** ✅ **Recommended**
   - Perfect match for frame canvas resolution
   - Excellent performance with 12 frames
   - Good quality for web display
   - File size: ~100-300KB per image

2. **1280 × 667 px** ⚡ **High Quality**
   - High-res version of the perfect aspect ratio
   - Still good performance 
   - Excellent for detailed screenshots
   - File size: ~200-600KB per image

3. **1920 × 1000 px** ⚠️ **Maximum Quality**
   - Ultra high-res - may impact performance
   - Best for high-detail work
   - File size: ~400KB-1.2MB per image

### **File Format**
- **JPEG** (.jpg) - Recommended for photographs and complex images
- **PNG** (.png) - Use for images with transparency or sharp edges
- **WebP** (.webp) - Best compression, but check browser support

## Image Preparation Tips

1. **Screenshot Capture**: Use browser dev tools to set viewport to 1280×667 for consistent web project screenshots with perfect aspect ratio
2. **Logo Projects**: Create mockups showing the logo in context (business cards, website headers, etc.)
3. **Consistency**: Use similar styling/backgrounds across images for a cohesive look
4. **Quality**: Ensure images are sharp and well-lit, especially for text/UI elements
5. **Optimization**: Compress images appropriately to balance quality and performance
6. **Aspect Ratio Tool**: Use image editing software to crop/resize to exactly 854×445 or 1280×667 for best results

## Fallback Behavior

If an image file is missing or fails to load, the system will automatically display a text-based version with the project title, description, and technologies in the widescreen format.
